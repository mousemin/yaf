### 目录说明
~~~
web
├─app                   应用目录
│  ├─controller         控制器
│  │  ├─Index.php       
│  │  └─...           
|  ├─exceptions         应用异常  
|  ├─facades            面板目录
|  ├─modules            模块
│  │  ├─User            用户模块
│  │  └─...             更多模块
|  ├─plugins            yaf插件
|  ├─services           service
|  ├─traits             traits
|  └─Bootstrap.php      yaf Bootstrap
├─config                配置
│  └─application.ini    yaf配置
├─extends               自定义类库文件 一般高度依赖yaf 不依赖一般使用composer加载
├─public                入口目录
├─composer.json         
~~~
